Техническое задание: Реализовать todo list (список дел).
Функционал: вывод списка задач, создание новой задачи, редактирование задачи, удаление задачи.
1. Основной компонент приложения - список задач. При открытии страницы должен показываться лоадер (задачи загружаются…). 
1.1 Если задач в базе нет, вывести сообщение об этом. 
1.2 Если есть задачи, список задач в виде таблицы (столбцы - id задачи, текст задачи, дата создания, статус, действия с задачей). Кнопка добавления новой задачи.
2. При нажатии на кнопку добавления задачи открывается модалка.
2.1 В модалке будет расположена форма создания новой задачи (текст задачи, дата создания, статус) + кнопка “добавить задачу” и кнопка “Отмена”.
2.2 Кнопка должна быть неактивной если какое-то из полей пустое.
2.3 При добавлении задачи сначала должен показываться лоадер (подождите, идёт добавление), а затем окно должно закрываться. В случае ошибки должно появляться сообщение.
2.4 Поле id должно генерироватся рандомно.
3. При наведении на строку задачи в таблице, возле текста задачи появляется иконка редактирования (карандаш в общем на ваш вкус и цвет).
3.1 При нажатии на кнопку редактирования открывается модальное окно с формой редактирования и кнопками  “Отмена” и “Сохранить”. 
3.2 При нажатии на “Сохранить” сначала показывается лоадер, затем окно закрывается и текст задачи в таблице меняется.
4. В последней колонке (действия с задачей) должна быть кнопка-иконка корзины и при наведении карандаш. 
4.1 При нажатии открывается модальное окно с подтверждением удаления. При подтверждении удаления сначала показывается лоадер, затем окно закрывается и задачи удаляется из списка.
5. При нажатии на шапку колонки весь список должен сортироваться (asc при повторном нажатии desc при следующем должна сбрасываться) и должна появляться соответствующая иконка применяемой сортировки если она есть.
5.1 Сортировка должна работать для всех колонок.
6. Добавить поиск или сортировку по колонкам.

7. Список задач должен храниться в LocalStorage или в идеале можете воспользоваться json server что бы поработать с запросами немного.
https://github.com/typicode/json-server -->
